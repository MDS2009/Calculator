# Calculator/Калькулятор
Калькулятор использует 2 перемнные.
# Функции:
"+" Сумма

"-" Вычетание

"*" Умножение

"/" Самое простое деление

"//" Это целочисленное деление

"%" Это остаток от деления

"**" Это возведение в степень
